# TASTE — the bar this project is judged against
George curates this. Claude checks every build against it BEFORE reporting done.

## Sites George rates (the bar)
- podium.global — preloader-gated opener, one choreographed reveal, nothing janks
- noxediem.ch — clean: one accent colour, huge type, hairlines, space, restraint per screen
- ponder.ai — the process/editor mechanic (studied frame-by-frame, _audit/ponder2)
- savor.it — line geometry, palette discipline (tokens in AIOS memory)
- lukebaffait.fr — gallery presence

## Hard rules learned on this build (violations = instant fail)
1. The page NEVER visibly parks. Scroll always answers. (Park detector must pass.)
2. No two sections share a layout. Same furniture twice = copy-paste feel.
3. Nothing designed at miniature scale. Styleframes are 1440×810, scaled down.
4. Show designs, never describe them. George picks with his eyes.
5. Archivo Expanded + Instrument Serif. Playfair is dead.
6. Feel > checklist. "Renders without errors" is not "good".
